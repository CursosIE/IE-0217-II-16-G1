Detailed Description

Clase Figura.

Clase base, posee como solamente 2 atributos: nombre y color; ademas de 2 funciones virtuales, las cuales son area y perimetro. Esta clase tiene 2 constructores, uno vacio (no recibe parametros) por default y el otro el caul recibe los atributos de las figuras en general.
